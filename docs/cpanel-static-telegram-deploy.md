# Быстрый деплой на cPanel без Node.js

Это самый простой вариант для текущего хостинга: сайт остается статическим, а заявки уходят в Telegram через `PHP`-файл `api/leads.php`.

## Сборка пакета

Для тестовой папки `/test`:

```bash
VITE_APP_BASE_PATH=/test npm run build:cpanel:static
```

После этого содержимое для загрузки появится здесь:

```bash
.deploy/cpanel-static/site
```

## Что входит в пакет

- статическая сборка сайта
- `api/leads.php`
- `api/telegram-config.php`, сгенерированный из локального `.env`
- `data/equipment.json`
- `.htaccess` и все нужные маршруты

## Куда загружать

Если тестируете на `https://arentex.by/test`, заливайте содержимое пакета в:

```bash
/public_html/test
```

Если выкладываете на основной сайт, заливайте в:

```bash
/public_html
```

## После загрузки

1. Распакуйте архив в нужной папке.
2. Подтвердите замену старых файлов.
3. Откройте сайт и отправьте тестовую заявку.

Если нужно отправлять заявки нескольким получателям, в `TELEGRAM_CHAT_ID` можно указать несколько id через запятую.

## Плюсы этого варианта

- не нужен `Setup Node.js App`
- не нужен `npm install` на хостинге
- не нужны переменные окружения в cPanel
- деплой почти такой же, как у обычной статики
